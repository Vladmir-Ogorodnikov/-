## 🔗 Подключение почтовых ящиков (Mail.ru и Gmail)

### 🟦 Mail.ru
1. Перейдите в настройки аккаунта: [Пароли и безопасность](https://account.mail.ru/security).
2. Прокрутите вниз до раздела **«Пароли для внешних приложений»** и нажмите **Добавить**.
3. Введите название (например, `Inbox AI`) и подтвердите действие вашим основным паролем.
4. Скопируйте сгенерированный 16-значный код 

### 🟥 Gmail
1. **Включите IMAP:** Откройте Gmail ⚙️ -> «Все настройки» -> Вкладка «Пересылка и POP/IMAP» -> Выберите **Включить IMAP** и сохраните изменения.
2. Перейдите в [Настройки безопасности Google](https://myaccount.google.com/security) (требуется включенная двухэтапная аутентификация).
3. Через поиск по настройкам найдите раздел **«Пароли приложений»**.
4. В выпадающем списке выберите *«Другое (Пользовательское название)»*, введите `Inbox AI` и нажмите **Создать**.
5. Скопируйте 16-значный код  

### 🚀 Авторизация в приложении
При регистрации или добавлении нового ящика в интерфейсе приложения укажите:
* **Логин:** ваш полный email-адрес (например, `example@mail.ru`).
* **Пароль приложения:** скопированный ранее код 
* **Сервис:** обязательно выберите соответствующий почтовый сервис из списка.
